# PlaywrightUsers

## Описание

Цель проекта: получение навыков в написание авто тестов на Java
на примере веб-сайта (<a href="http://users.bugred.ru/">users</a>).

## Технологии

- Java
- JUnit
- Playwright
- Allure
- Gradle
